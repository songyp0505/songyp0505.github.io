发表于 2022-03-06 更新于 2022-06-30 分类于 周记

开学了开学了开学了

本周

这周主要都是在上网课和赶路了，在家上网课的时候虽然没有线下上课的场地约束，但是也得专门空出时间，可是又会跟各种事冲突，所以才热切地盼望着开学。

上半周上课上得比较多，包括不同老师布置的一些作业，没怎么精读论文，周三开始赶路，在我姐家住了两天周六乘机回来，开学的疲惫。

回过头看看这一周，好像啥也没干，但是又好像每天都很忙。

下周

  1. 把王老师寒假前布置的读书笔记整理整理。

  2. 整理出具体的小论文题目或者限制范围较窄的主题。

  3. 根据题目或者主题继续精读论文。

  4. 把上学期剩下的《研究是一门艺术》下半部分读完。
